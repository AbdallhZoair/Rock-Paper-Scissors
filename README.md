Rock, Paper, Scissors ✊📄✂️

A simple command-line Rock Paper Scissors game in Python, with ASCII art for each choice.

What it does


The computer picks Rock, Paper, or Scissors randomly.
You type your choice.
Type Help at the start if you want to see the rules first.
The game prints both choices as ASCII art, then tells you who won.


How to run it

bashpython3 rock_paper_scissors.py

Concepts practiced


random.randint() to make the computer's choice
List indexing (choose[0], choose[1], choose[2])
Nested if / elif / else for comparing choices
.capitalize() to make user input case-insensitive
